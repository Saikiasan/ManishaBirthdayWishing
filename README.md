# Manisha's 24th Birthday Website

A modern, responsive single-page birthday wishing website for Manisha, celebrating her 24th birthday in 2026.

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices
- **Modern UI**: Beautiful gradient animations, floating elements, and smooth transitions
- **Fast Loading**: Lazy loading for images and videos for optimal performance
- **Interactive Gallery**: Clickable photo gallery with lightbox feature
- **Auto-pause Videos**: Automatically pauses other videos when one is played
- **Smooth Animations**: Intersection Observer-based reveal animations

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Poppins)

## How to Use

1. Open `index.html` in any modern web browser
2. Browse through the birthday message
3. View the photo gallery (click to enlarge)
4. Watch the video memories
5. Share with friends!
